# Mariam Sewify - Portfolio Website

A modern, responsive portfolio website showcasing web development skills and projects.

## 🎯 Project Overview

This is a professional portfolio website built for **Mariam Sewify**, a web developer from Alexandria. The website features a clean, modern design with smooth navigation and interactive elements to showcase development work and skills.

## ✨ Features

- **Responsive Design** - Fully responsive layout that works on desktop, tablet, and mobile devices
- **Smooth Navigation** - Sticky navbar with smooth scrolling to different sections
- **Hero Section** - Eye-catching introduction with call-to-action buttons
- **About Section** - Personal introduction, contact information, and tech stack display
- **Projects Showcase** - Portfolio projects with descriptions, technology tags, and live links
- **Contact Section** - Get in touch form for inquiries
- **Modern UI** - Beautiful gradient backgrounds, hover effects, and animations

## 🛠️ Tech Stack

- **HTML5** - Semantic markup for structure
- **CSS3** - Custom styling with advanced features (gradients, animations, flexbox)
- **JavaScript** - Interactive functionality
- **Bootstrap 5** - Responsive grid system and components
- **Bootstrap Icons** - Icon library for visual elements
- **Google Fonts** - Inter font family for modern typography

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required - everything is bundled

### Installation

1. Extract or clone the project files
2. Open `index.html` in your web browser
3. The website will load with all styling and functionality ready to use

## 💡 Customization

To customize this portfolio:

1. **Update Personal Information:** Edit name, bio, email, and location in the HTML
2. **Change Colors:** Modify the color variables in the CSS (e.g., #0d6efd for primary color)
3. **Add/Remove Projects:** Edit the projects section with your own project cards
4. **Update Links:** Replace placeholder social media and project links with real URLs
5. **Add images:** Place your project screenshots in the images folder and update image paths
